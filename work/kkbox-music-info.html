WEB UI DESIGN, 2016

KKBOX 
Music Info

KKBOX is Asia's leading music streaming service with over 10 million users. In order to provide seamless experience from Web to APP, we decided to redesign the “Music Info” feature on KKBOX mobile website.

The Challenge

Users were confused

The “Music info” feature within KKBOX helps users discover new music by showing them new album releases and curated playlists, and allows them to install or open KKBOX app to enjoy music with just one single click.

The previous version of website had usability problems that made it challenging for users to listen to music quickly. The problem existed in not only Taiwan but also all the service regions. Some major issues were:

Cognitive load

  • Confusing call-to-actions
  • Overloaded information
  • Poor readability/ tapability

Lack of consistency

Inconsistent user interface across platforms.

My Role: UI Designer
Deliverables: High-Fidelity Mockups, InVision Prototypes, Zeplin Specs, and UI Kit.

Goal

To provide seamless experience from Web to APP

The main motivation behind this redesign is to optimize usability, improve conversion rates, and create a seamless, intuitive experience that enables users to navigate end enjoy music more easily and quickly.

  • Simplify the UI to provide the only needed information.
  • Provide consistent user experience across platforms, especially web to app.
  • Optimize readability to improve browsing experience on mobile.
  • Facilitate the multilingual user interface.
Design Solution

Optimize screen real estate

Positioning a single prominent CTA and primary content in the center to improve conversion rate.

Design Solution

Flexible, modular and component-based design

Using wide rows instead of narrow columns. Leaving more space for translation of product title and artist name to ensure users' readability on mobile.
